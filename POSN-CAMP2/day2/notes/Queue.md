# QUEUE

## DEFINITION
- FIFO [ First in First out] data structure
- 2 important sides
1. front
2. back

## IMPORTANT OPERATIONS

|OPERATION|FUNCTION|
|-|-|
|.push_back()|or enqueue, adds to the back of the queue|
|.pop_front()|removes the frontmost value of the queue|
|.back()|gets the backmost value of the queue|
|.front()|gets the frontmost value of the queue|
