# DATA STRUCTURES

> [!NOTE]What are data structures
> data structures are more convenient methods of storing data for specific needs

---
### LINEAR DATA STRUCTURES

- Stores data in a single line (linearly)
- Some examples include:

|DATA TYPE|DEFINITION|
|-|-|
|ARRAY|array of something|
|STACK|similar to array but only the top is accessable|
|Queue|outputs in the front and inputs into the back|
|Linked List|similar to array but each node is a different object that has a pointer pointing to the next|
