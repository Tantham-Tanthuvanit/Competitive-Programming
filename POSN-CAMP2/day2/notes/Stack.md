# STACK

## definition
- LIFO data struct [ LAST IN FIRST OUT ]
- Only 1 usable side [ top ]

## important operations
|OPERATION|FUNCTION|
|-|-|
|.top()|returns the topmost value|
|.push(x)|adds x to the top of the stack|
|.pop()|removes the top of the stack|
|.empty()|check if stack is empty|

> [!WARNING] ALWAYS CHECK IF STACK IS EMPTY BEFORE DOING .top() or .pop()
