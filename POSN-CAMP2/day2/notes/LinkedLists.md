# LINKED LISTS

> [!NOTE] doubly linked list is also a thing
> where theres 2 pointers, [next] and [prev]

## DEFINITION
- Nodes that constantly point to the next node using a pointer

```[NODE 1] -> [NODE 2] -> [NODE 3]```

- First node is called the head
- Usually has 2 values [value] and [*next]
- Similar to array but can be manipulated in more complex ways

## BASIC OPERATIONS
|OPERATION|DEFINITION|
|-|-|
|Insert at front|insert a node infront|
|Delete at front|delete the frontmost node|
|Traverse|travel through a linked list|
|Search|search for a specific node|
|Insert at position|insert a node at a specific position|
|Delete at position|delete a node at a specific position|
